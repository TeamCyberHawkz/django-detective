# 🕵️ Django Detective

> Real-world Django challenges inspired by production incidents, performance bottlenecks, security flaws, and business logic bugs.

Welcome to **Django Detective**.

Most coding challenges test whether you can write code.

Django Detective tests whether you can **think like an engineer**.

Each challenge simulates a real-world issue that engineering teams face in production environments. Your mission is to investigate the symptoms, identify the root cause, and submit a professional solution.

Whether you're a student, intern, junior developer, or experienced engineer, these challenges are designed to help you evaluate the skills that matter most in software development.

---

## 🎯 How It Works

Each challenge lives in its own repository.

Every repository contains:

* A Django application
* A realistic problem statement
* Production-style clues
* Hidden bugs or inefficiencies
* Requirements for submission

Your goal is to:

1. Investigate the issue.
2. Identify the root cause.
3. Implement a fix.
4. Add tests where appropriate.
5. Submit a Pull Request.
6. Explain your reasoning.

## 📋 Submission Process

### Step 1: Fork the Challenge Repository

### Step 2: Create a Branch

### Step 3: Implement Your Solution

Your solution should:

* Fix the identified issue
* Follow Django best practices
* Keep code clean and maintainable
* Include tests when applicable

### Step 4: Commit Your Changes

### Step 5: Push Your Branch

### Step 6: Open a Pull Request

Create a Pull Request against the challenge repository.

---

## 📝 Pull Request Template

Please include the following sections in your PR:

### Problem

What issue did you identify?

### Root Cause

Why did the issue occur?

### Solution

How did you fix it?

### Testing

How did you verify the fix?

### Additional Improvements

Did you find any other issues or optimizations?

## 🤝 Why We Built This

Traditional coding assignments often focus on solving isolated problems.

Real engineering is different.

Engineers investigate ambiguous issues, work with incomplete information, analyze production behavior, communicate findings, and deliver maintainable solutions.

Django Detective is designed to simulate that experience and test the Developers who can code real world solutions.  .
